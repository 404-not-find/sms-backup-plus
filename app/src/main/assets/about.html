Author: Jan Berkel (and many others, see credits below)

Automatically backs up your SMS/MMS and call logs to Gmail or any other IMAP server. Can also restore SMS / call logs after they have been backed up.

New in this release:

  • Runtime permissions on Android 6.0+. It is recommended to run a manual backup after upgrade.
  • Improved auto backup reliability (thanks Mads Andreasen for initial JobManager implementation)
  • Fixed browser authentication
  • Improved restore experience (thanks Anton Keks/angryziber)
  • Modernized UI
  • Updated notification images
  • Updated Korean translation

See changes on github for the full list.

Information

Please visit the SMS Backup+ github page for more information. Also make sure to check the FAQ before reporting a bug.
